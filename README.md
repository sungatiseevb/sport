## Что нужно сделать

1. Открой C++ Builder XE
2. Создай новый проект: `File → New → VCL Forms Application`
3. Для каждой следующей формы: `File → New → Form`
4. Расставь компоненты на каждой форме (см. ниже)
5. Скопируй код из `.cpp` и `.h` файлов
6. Настрой порядок форм: `Project → Options → Forms` — Form1 должна быть первой
7. Собери проект: `Run → Build`
