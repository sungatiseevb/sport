# sport


## Содержание

В проекте присутствуют следующие файлы:

- `Unit1.cpp`, `Unit1.h` — первые задания
- `Unit2.cpp`, `Unit2.h` — вторые задания
- `Unit3.cpp`, `Unit3.h` — третьи задания
- `Unit4.cpp`, `Unit4.h` — четвертые задания
- `Unit5.cpp`, `Unit5.h` — пятые задания

## Сборка и запуск

Сборка и запуск выполняются стандартным компилятором C++ (например, `g++`).

Пример сборки:

```bash
g++ -std=c++17 Unit1.cpp -o unit1
./unit1
```

Использование, копирование и изменение кода в этом репозитории осуществляется согласно выбранной вами лицензии.
